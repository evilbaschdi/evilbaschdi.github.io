Below I've listed my current repositories of tools that make my life more easy.

  • DisplayRotation get_app

    Tool for rotating a display directly from the windows notification bar
    because of AMD or Nvidia gpu drivers do not offer shortcuts like Intel drivers to do that.

  • ListFilesByDate get_app

    This application provides the possibility to list files that where change before or after a special date.

  • Folders2Md5 get_app

    A little tool I wrote to generate checksum files of type md5 for each file in a folder and its subdirectories.
    It's also possible to trigger this tool through command line / scheduled task to generate the checksums in an interval to provide them for ftp downloads for example.

  • Movies get_app

    An xml-based application to manage my private movie collection.

  • VmMachineHwVersionUpdater get_app

    Using the vmware player it's not possible to upgrade the hardware version of a machine like it is with vmware workstation.
    After upgrading to player version 7 I was tired of editing each vmx file separately. So I decided to create a little application to do this for me this time and for the next upgrade.

  • TestDataGenerator get_app

    A small project to generate test data for developing purposes.

  • IISWorkerProcessLister get_app

    Sometimes this application should show running IIS processes in the task tray; Unstable.

  • WinSPCheck get_app

    Sometimes ago this application was developed to show if a windows os has the latest service packs installed. Since Microsoft has stopped developing service packs for windows and changes the build lab string with every second update I don't know if I should develope this furthermore.
